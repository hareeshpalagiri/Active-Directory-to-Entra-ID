# Active Directory to Microsoft Entra ID

A comprehensive security reference for **On-Premise, Hybrid, and Cloud** environments.  
This repository provides a structured guide to Identity and Access Management (IAM), Active Directory, and Microsoft Entra ID, including attack techniques, security hardening, and real-world labs.

- **Section-02-Active-Directory** → AD Architecture, Forests, Domains, OUs, GPOs
- **Section-03-AD-Auth-Protocols** → Kerberos, NTLM, LDAP, Smart Cards
- **Section-04-AD-Security** → Tiered Admin Model, LAPS, Credential Guard
- **Section-05-AD-Attacks** → Kerberoasting, Golden Ticket, DCSync, BloodHound

- **Section-07-Entra-ID-Security** → Conditional Access, MFA, Identity Protection
- **Section-08-Hybrid-Identity** → Entra Connect, ADFS Federation, Seamless SSO
- **Section-09-Monitoring-Logging-IR** → Event IDs, Defender for Identity, Sentinel
- **Section-10-Labs-Scenarios** → Hands-on labs, simulations, interview Q&A
